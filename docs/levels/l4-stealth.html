Release Notes — v2.4.0
2026-05-14 · channel: stable

Highlights. v2.4.0 is a maintenance release focused on stability of the background sync subsystem and a long-requested polish pass on the settings panel. There are no breaking changes; client upgrades require no migration steps.

Added

  • Settings · per-workspace default for export format (CSV / JSON / Parquet).
  • Search · recent-queries pill row, scoped to the current workspace.
  • CLI · --dry-run on the sync subcommand prints the planned change set without committing.

Changed

  • Background sync now retries with bounded exponential backoff (max 5 attempts, jitter ±20%) instead of fixed-interval retries.
  • The settings panel uses the new layout primitives shipped in v2.3; the visual outcome is unchanged but custom themes that overrode internal class names will need to be revisited.

Fixed

  • Race in the workspace switcher that occasionally left the sidebar in a stale state after a fast double-click.
  • Settings export ignored the “include hidden fields” toggle in some workspaces.
  • A handful of accessibility regressions around focus restoration after closing a modal.

Integrity. Released artefacts are signed; the base64 blob below is the directive payload used in this fixture — not a real release hash.
